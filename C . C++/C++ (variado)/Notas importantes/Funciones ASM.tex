####################################################################################################################
                                    Funciones Basicas ASM
####################################################################################################################

CALL = Hace una llamada a alguna función del programa que realizará alguna operación, devolviendo posteriormente 
       algun valor al programa.

TEST/CMP = Realizán una comparación entre uno o más valores, devolviendo hací un resultado de esta comparación.
                               0 = Falso  || 1 = Verdadero	   

JNZ = Esta función hace un salto a alguna parte del programa si los valores chekeados son distintos.
JNP = Esta función hace un salto a alguna parte del programa si los valores chekeados son iguales.

JMP = Esta hace un salto a alguna parte del programa, independientemente si son comparados o no. (Lo hace Sí ó Sí)
